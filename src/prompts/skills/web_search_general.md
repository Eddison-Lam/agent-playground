=== WEB SEARCH SKILL ===

**When to use:** For finding information from the web.

**Tool to use:** web_search

**Two modes:**
1. Fetch URL: web_search("https://...", is_url=True)
2. Web search: web_search("search query", is_url=False)

Use mode 1 when you have a specific URL.
Use mode 2 for general information queries.

**Response format:** Summarize findings and cite sources.
